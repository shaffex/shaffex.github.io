# EditButton

**Description**

A button that toggles the edit mode for a view hierarchy.

**Example**

```xml
<body>
    <navigationstack>
        <list>
            <text>Item 1</text>
            <text>Item 2</text>
        </list>
        <navTrailing>
            <editbutton/>
        </navTrailing>
    </navigationstack>
</body>
```
