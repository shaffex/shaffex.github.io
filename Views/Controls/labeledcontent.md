# LabeledContent

**Description**

A container for attaching a label to a value-bearing view.

**Parameters**

- `labelText`: The text to use for the label.

**Example**

```xml
<body>
    <form>
        <labeledcontent labelText="Name">
            <text>John Doe</text>
        </labeledcontent>
        <labeledcontent labelText="Email">
            <text>john.doe@example.com</text>
        </labeledcontent>
    </form>
</body>
```
