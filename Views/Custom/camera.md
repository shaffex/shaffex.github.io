# Camera

**Description**

A view that displays a camera interface for taking photos.

**Parameters**

- `sourceType`: The source for the camera view. Can be `camera`, `savedPhotosAlbum`, or `photoLibrary`.

**Example**

```xml
<body>
    <camera sourceType="camera"/>
</body>
```
