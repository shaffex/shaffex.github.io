# YouTubeVideoView

**Description**

A view that displays a YouTube video.

**Parameters**

- `videoId`: The ID of the YouTube video to display.

**Example**

```xml
<body>
    <youtubevideoview videoId="uBoijwzuY5s"/>
</body>
```
