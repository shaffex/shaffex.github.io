# ForEach

**Description**

A structure that computes views on demand from an underlying collection of identified data.

**Parameters**

- `repeatCount`: The number of times to repeat the child views.

**Example**

```xml
<body>
    <vstack>
        <foreach repeatCount="5">
            <text>Hello</text>
        </foreach>
    </vstack>
</body>
```
